## Installation

To install fairsharer from the GitHub repository, run:

```bash
git clone https://github.com/VipusinyVijayakumar/VipusinyVIjayakumar-fairsharer.git
cd fairsharer
python -m pip install .
# or on macOS:
python3 -m pip install .
